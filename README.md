# QinTechBoard

## QinTechBoardとは

Qinで共有されている技術をまとめたサービスです。

## 目的

新規入国者がサロン内で利用されている技術の用語を探しやすくすることで、
用語の共通化を促進し円滑なコミュニケーションを取れるようにする！
